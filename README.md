# ⛳ GolfLine – Howth Golf Club GPS

A progressive web app (PWA) GPS caddie for **Howth Golf Club, Dublin**.

## Features

- 📍 **Live GPS distance** to pin with animated ring display
- 🗺 **Satellite course map** with real OSM fairway/green/bunker overlays
- 📋 **Full scorecard** with colour-coded scoring and +/- vs par
- ⛳ **Auto hole advance** — prompts when you reach the green
- 🏌️ White / Yellow / Red tee distances for all 18 holes
- Dashed routing line from your position to the pin

## Install on Android

1. Open the live URL in **Chrome on Android**
2. Tap ⋮ menu → **Add to Home Screen**
3. Allow location when prompted

## Install on iPhone

1. Open the live URL in **Safari**
2. Tap the Share button → **Add to Home Screen**

## Live URL

👉 `https://YOUR-USERNAME.github.io/howth-golf-gps`

## Course Data

Scorecard data sourced from the official Howth Golf Club scorecard (White/Yellow/Red tees, par 71).  
Course map data loaded live from [OpenStreetMap](https://openstreetmap.org) via the Overpass API.
